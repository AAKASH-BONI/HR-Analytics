## Overview

The HR Analytics Dashboard is an end-to-end Power BI project designed to analyze workforce data and uncover key insights related to employee attrition, demographics, compensation, and job satisfaction.

This dashboard transforms raw HR data into interactive visualizations that help organizations identify retention risks, understand workforce trends, and support data-driven HR decision-making.

Built with a modern dark-themed UI and intuitive navigation, the report enables users to explore insights dynamically through slicers, KPI cards, and drill-down visuals.

## Objectives

- Analyze employee attrition patterns and key contributing factors
- Monitor workforce demographics such as age, gender, and department distribution
- Evaluate salary trends and job satisfaction levels
- Provide actionable insights to improve employee retention and HR strategy
- Design a user-friendly dashboard for business stakeholders

## Dataset

The dataset used in this project is confidential and cannot be shared publicly. It reflects a realistic HR data structure including employee demographics, compensation, and attrition metrics for analytical purposes.

## Key Features

- Interactive KPI cards (Attrition Rate, Avg Age, Avg Salary, Satisfaction Score)
- Department-wise and gender-based attrition analysis
- Age group segmentation for workforce insights
- Job role and satisfaction correlation analysis
- Dynamic slicers for department-level exploration
- Clean dark UI dashboard for improved readability

## Business Impact

This dashboard can help HR teams:

- Identify high attrition departments and roles
- Detect workforce demographic patterns
- Improve retention planning and employee engagement strategies
- Support leadership with data-driven workforce decisions

## Conclusion

This project showcases an end-to-end HR analytics solution built in Power BI, highlighting attrition trends and workforce insights through interactive visuals. It demonstrates the ability to combine data analysis, visualization, and business understanding to support data-driven HR decisions.
